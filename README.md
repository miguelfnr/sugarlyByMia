# Sugarly By Mia 🧁

Este é um projeto de website para a **Sugarly By Mia**, desenvolvido com **Next.js 15**, **Tailwind CSS** e **Lucide Icons**.

## Como fazer o Deploy para a Vercel

1.  Cria um novo repositório no teu **GitHub**.
2.  Move os ficheiros desta pasta para o novo repositório ou faz `git init` dentro desta pasta.
3.  Faz login na [Vercel](https://vercel.com).
4.  Clica em **Add New** > **Project**.
5.  Importa o repositório do GitHub.
6.  A Vercel detetará automaticamente que é um projeto Next.js. Clica em **Deploy**.

## Desenvolvimento Local

```bash
npm install
npm run dev
```
